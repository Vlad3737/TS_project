# Time Series Ensemble Project

Проверка гипотезы: ансамбли моделей дают более стабильный прогноз, чем одна "лучшая" модель.

## 📊 Результаты

| Модель | SMAPE | std SMAPE |
|--------|-------|-----------|
| SeasonalNaive | 28.25 | 4.67 |
| CatBoost | 30.86 | 5.85 |
| **Ensemble (80/20)** | **27.50** | **4.82** |

✅ **Гипотеза подтверждена**: ансамбль улучшил качество на 2.7% и снизил нестабильность

## 🚀 Быстрый старт

```bash
# 1. Установка
pip install -r requirements.txt

# 2. Загрузка данных и запуск экспериментов
python run_experiment.py

# 3. Просмотр отчета
jupyter notebook results/analysis_results.ipynb
```

## 📁 Структура
- `src/` - код моделей и ансамблей
- `results/` - отчет с графиками
- `data/` - данные M4 Hourly

## 🛠 Требования
Python 3.9+, библиотеки в requirements.txt
